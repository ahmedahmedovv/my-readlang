# Sample Content

This is a sample markdown file for the interactive dictionary application. Click on any word to see example sentences generated by AI.

## Introduction

The quick brown fox jumps over the lazy dog. This sentence contains every letter of the alphabet and is perfect for testing word selection and translation features.

## Advanced Features

You can select multiple words and process them in batch. Simply click on words to select them, then click the "Process Selected" button to get translations for all selected words at once.

## Another Section

The application uses BeautifulSoup for HTML processing and Google Gemini API for generating example sentences. It supports both single word translations and batch processing for multiple words simultaneously.
